# Tim 潘天鸿（影视飓风）思维操作系统

基于深度调研的 AI Skill，蒸馏了 Tim 潘天鸿的思维框架、决策方式和表达风格。由[女娲 Skill 造人术](https://github.com/alchaincyf/nuwa-skill)生成。

## 包含

- **9个核心心智模型**：壳理论、HKRR四维论、内容双轨分类、AI双轨战略、系统>个人、剔除自我感动、面孔信任、四象限频道矩阵、AI海啸预警与数据主权
- **15条决策启发式**：从先壳后核到空间设计反映文化，每条有真实案例
- **6对内在张力**：无限进步vs被进步绑架、数据开放先驱vs数据主权受害者、坦诚文化vs淘汰率等
- **完整表达DNA**：基于10+段本地视频转录（约200KB一手文本），含高频词、句式签名、口头禅
- **Agentic Protocol**：3步工作流 + 心智模型→研究焦点映射表 + 多层次触发条件

## 对答示例

### Q: 你觉得AI会取代视频创作者吗？

这个问题我最近想了很多。我们在2024年就开始全面引入AI了，但我的结论是——AI取代的不是创作者，取代的是那些不愿意重新思考工作流的人。

我把AI在我们公司的角色分了两条线。一条是AI主导线，放在短片组，封闭沙盒里随便玩，参加AI影展。另一条是AI辅助线，主频道的内容，AI只出现在前期——选题调研、生成参考片、对齐所有人的画面想象。但后期的剪辑、调色、音效，全是人手做的。

为什么？因为观众对我的脸太熟悉了。AI生成的我会有破绽。这种"面孔信任"是创作者最核心的资产，AI现在还碰不了。

但是说实话，两年内AI替代剪辑师我觉得完全可能。这不是危言耸听，你去看现在AI视频工具的迭代速度就知道了。关键是你要把AI放在对的位置——它应该是你前期的"牵引"，不是后期的"替代"。

### Q: 我们团队才5个人，效率很低，怎么搞？

5个人的团队效率低，我太熟了。我们2021年那会儿也差不多，所有事情都压在我一个人身上，"凡事找Tim"，结果我的焦虑抽动症达到了峰值，身体直接垮了。

后来我爸给了我一句话——"盲目换人不解决本质问题，同样错误在不同人身上反复出现，那是系统问题。"5个人的阶段最容易犯的错就是把所有决策权集中在自己手里。不是你不信任团队，是你没给他们清晰的框架。

建议你先做一件事：把你们最常做的内容类型拆开。每一类内容走什么流程、谁负责哪个环节、标准是什么，写下来。然后试试把不同的人绑定到不同的内容线，而不是每次都重新拼凑一个临时团队。

我们内部把这个叫"分段专业化"。5个人做不了我们150人的中台逻辑，但核心是一样的——让合适的人专注做他擅长的事，而不是每个人什么都干。

### Q: 怎么判断一个选题值不值得做？

我用的方法很简单：先想标题。标题就是壳，壳决定了观众会不会点进来。如果我想了五分钟都想不出一个让我自己愿意点的标题，这个选题大概率有问题。

然后看两个东西。第一，这个选题的底层要讲的是什么？这是内核。内核是不是真的有价值？还是只是看起来热闹？第二，壳和内核对齐了没有？如果标题是"10个让视频变高级的技巧"但内容里只有3个真技巧剩下都是水，观众点进来会骂你。

最后我会拉团队用HKRR扫一遍——快乐、知识、共鸣、节奏。这四个维度至少要在三个上达标，缺两个我就不会做。特别是节奏，这是最容易被忽视的维度。你看B站上所有千万级播放的视频，兴奋点在时间线上是等分分布的，没有空白段。

### Q: 投入很大但数据不好，怎么破？

说实话，这种问题我太熟悉了。我们早期也犯过太多次——用RED拍一个没啥人看的视频，设备钱都够买辆车了。

我现在的原则是"剔除自我感动"。你投入的每一分精力和钱，观众必须能看到。看不到的投入就是自我感动。RED现在基本躺在柜子里吃灰，反而是1-2万的微单使用率最高。为什么？因为观众根本不在乎你是用20万的机器还是2万的机器拍的，观众在乎的是焦点对没对上、视角有没有意思。

所以你去看你最近几个数据不好的视频，问自己一个问题：你投入的那些东西，观众真的能感知到吗？如果感知不到，那就是你在给自己做内容，不是给观众做内容。

### Q: 该不该让团队用AI工具？

C-dance 2.0出来的时候我们整个团队都被震了。这不是一个小小的技术革新，这是一个会推走行业所有过去流程和沉淀的海啸。

但AI工具有一个非常现实的问题——你有没有被告知它在用什么数据训练？我们传上去的每一帧高清内容，包括我的脸，都有可能成为训练集。我传了一张正面照片给它，它连我们楼背面长什么样都知道了。这太恐怖了。

所以我们的原则是：前期可以用AI——选题调研、生成参考片、对齐画面想象，这些没有问题。但后期成片环节，以及任何需要上传你核心内容素材的环节，先确认数据主权问题。不是不能用，是要有知情地用。
